About Squad Tools

It is a collection of useful infographics, guides, stats, videos and more. I created this site to help educate new players on finer details about Squad that many expect you to know, but the tutorial does not cover. Beyond teaching, I also aim to create advance guides, and reference material to help support the more experience Squad community too. Squad tools is an open source, allowing anyone to contribute updates / changes on the GitHub!

Using The Website

Navigate to the NAV bar at the top of the page, from there you can select different pages. Currently tank pages include weak spots and vehicles stats. I plan to expand this for IFVs in the future. There is also factions, here you can learn about their equipment/vehicles and how to PID them.

Open Source <3

If you notice outdated info, or errors, this site is open source and can be found @ github.com/tempusowl/squadtools from there you can find all the source files and guides on how to contribute!
